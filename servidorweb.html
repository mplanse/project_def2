    Documentacion de la instalacion del Servidor Web en Ubuntu Server

    En este apartado de la pagina os enseñaremos la documentacion del proceso de instalacion del Servidor Web en Ubuntu Server

    Paso 1:Primero de todo hemos de usar estos comandos sudo apt update sudo apt install apache2 para instalar apache2

    Paso 2: La configuración de host virtual de Apache predeterminada www.example.comy configuraremos la nuestra en Html.example.cat, Entonces comenzamos creando una carpeta para nuestro nuevo sitio web /var/www/ejecutando.Y luego ponemos para finalizar el comando sudo mkdir /var/www/gci/

    Paso 3: Vamos a nuestro directorio recién creado y creemos uno escribiendo:

    cd /var/www/gci >

    Paso 4: Comenzamos este paso yendo al directorio de archivos de configuración:

    Pegamos este comando cd /etc/apache2/sites-available/ para seguir con la instalacion .

    Ahora editamos el archivo de configuraciónl:sudo nano gci.conf

    Paso 5: El archivo predeterminado no viene con una ServerNamedirectiva, por lo que tendremos que agregarlo y definirlo agregando esta línea debajo de la última directiva:

    Esto garantiza que las personas lleguen al sitio correcto en lugar del predeterminado cuando escriben gci.example.cat.

    sudo cp /etc/b9ind/db.local /etc/biind/db.midominio.local

    Paso 6: Después de configurar nuestro sitio web, debemos activar el archivo de configuración de hosts virtuales para habilitarlo. Lo hacemos ejecutando el siguiente comando en el directorio del archivo de configuración: sudo a2ensite gci.conf

    .

    sudo a2ensite gci.conf

    Paso 7:En el ultimo paso si se a hecho todo bien debereiamos ver lo siguiente

    service apache2 reload root@ubuntu-server:/etc/apache2/sites-available#

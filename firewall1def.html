    Documentacion de la creación de un Firewall con iptables

    En este apartado de la pagina os vamos ha ensñar como nuestro equipo a logrado crear un firewall con unas normas predefinidas.

    Paso 1: Identificar que normas nos pedia el cliente.

    Para saber como debemos hacer el firewall es obligatorio saber que quiere permitir y denegar el cliente

    Paso 2: Procedemos a crear las normas

    Para cada regla mostraremos como lo hemos hecho para que haga efecto

    Norma 1: Permitir la navegacion web segura

    iptables -A INPUT -p tcp -m multiport --dports 80,443,3000,5000,22,21 -m conntrack --ctstate NEW,ESTABLISHED -j ACCEPT

    Norma 2: Permitir conexion remota

    iptables -A INPUT -p udp --dport 53 -m conntrack --ctstate NEW,ESTABLISHED -j ACCEPT

    Norma 3: Permitir conexion a DNS externos

    iptables -A OUTPUT -p tcp -m multiport --sports 80,443,3000,5000,22,21 -m conntrack --ctstate ESTABLISHED -j ACCEPT

    Norma 4: Permitir conexiones ftp

    iptables -A OUTPUT -p udp --sport 53 -m conntrack --ctstate ESTABLISHED -j ACCEPT

    Norma 5: Denegar el resto de conexiones

    iptables -A INPUT -j DROP iptables -A OUTPUT -j DROP

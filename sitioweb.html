    Paso 1: Creamos el indice general en el que se encuentra el "nav" y elementos introductorios del proyecto

    Para esto creamos el elemento general nav, con subelementos que crearemos usando "ul" y "li"

    Paso 2: Dentro de cada elemento del nav crearemos html

    Dentro de cada elemento subiremos un html para la documentacion de los apartados, pero es facil ya que solo crearemos uno, especificaremos elementos de titulo subtitulo y texto para de alguna manera tener una plantilla predefinida para todos los apartados

    Paso 3: La tabla de presupuesto

    Para crear una tabla para el presupuesto usaremos el comando table, junto con td, tr, th .No podemos olvidar definir clases con el comando class ya que asi luego los estilos seran mas faciles de implementar

    Aqui podemos ver un trozo de codigo de la tabla, en la que podemos ver que usamos td tr th cada uno con su clase correspondiente
